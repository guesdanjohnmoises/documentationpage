HTML

HTML stands for Hypertext Markup Language.

It is the standard markup language for creating web pages.

HTML elements are the building blocks of HTML.

<html> <head> <body>
  • <html> - defines the document
  • <head> - contains metadata
  • <body> - contains content
CSS

CSS stands for Cascading Style Sheets.

It is used to style and layout web pages.

CSS controls the layout, fonts, colors, and spacing of HTML elements.

body {} h1 { color: blue; } p { margin: 20px; }
  • Selectors
  • Properties
  • Values
JavaScript

JavaScript is a programming language used to create dynamic and interactive web content.

It can manipulate HTML and CSS to update the page content without reloading.

JavaScript runs in the browser, making websites more interactive.

console.log("Hello, World!"); let x = 10; document.getElementById("demo").innerHTML = "Hello JavaScript!";
  • Variables
  • Functions
  • Events
API

API stands for Application Programming Interface.

An API allows one software application to interact with another.

APIs are used to connect web services and enable communication between different platforms.

fetch('url').then(response => response.json()); axios.get('url').then(response => { ... }); XMLHttpRequest.open('GET', 'url', true);
  • HTTP Methods
  • Endpoints
  • Authentication
Tools

Tools are essential for web development and testing.

Some common tools include code editors, version control, and testing frameworks.

They help developers streamline their workflows and improve productivity.

VS Code Git Jest
  • Version Control (Git)
  • Text Editors (VS Code, Sublime Text)
  • Testing Frameworks (Jest, Mocha)
